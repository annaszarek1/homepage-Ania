Notatki HTML

HTML = HyperText Markup Language

Lekcja: W02E03 HTML I
-> To szkielet strony internetowej.
-> Opakowuje i opisuje treść strony
-> Mówi przeglądarce o strukturze dokumentu (i o tym co ma wczytać)
-> Posiada swoją zdeﬁniowaną składnię
-> Nie jest językiem programowania, jest językiem znaczników (Markup Language)


Tagi/Znaczniki/Elementy HTML

Lekcja: W02E03 HTML I
"html" & "/html" - opisuje cały dokument HTML
"head" & "/head" - głowa dokumentu
"body" & "/body" - ciało dokumentu
"main" & "/main" - główna część ciała dokumentu
"section" & "/section" - sekcja dokumentu
"header" & "/header" - nagłówek dokumentu/sekcji
"footer" & "/footer" - stopka dokumentu/sekcji
"h1"-"h6" & "/h1"-"/h6" - tekstowy nagłówek tytułowy dokumentu / sekcji
"p" & "/p" - paragraf
"br" - znak przejścia do nowej linii

Lekcja: W02E04 CSS I
"link" - tag/znacznik/element HTML, który można dodać w sekcji "head" aby użyć stylów znajdujących się w osobnym pliku (np. styles.css),
(adres/nazwę pliku .css definiujemy atrybutem "href")

Lekcja: W03E03 HTML II
"article" & "/article" - samodzielna treść
"b" & "/b" - pogrubienie tekstu (bold), nie ma znaczenia semantycznego (znaczeniowego) - tylko wizualne
"strong" & "/strong" - pogrubienie tekstu, ma znaczenie semantyczne, jest zauważalne dla czytnika lub robota, podkreśla wysoką istotność i poważność frazy w nim zawartej
"i" & "/i" - kursywa (italics), nie ma znaczenia semantycznego (znaczeniowego) - tylko wizualne
"em" & "/em" - kursywa, ma znaczenie semantyczne, jest zauważalne dla czytnika lub robota, kładzie mocniejszy nacisk na zawartość
"a" & "/a" - linki/łącza wewnątrz dokumentu (do wewnętrznych i zewnętrznych treści)
"img" - definiuje obrazek (plik graficzny)

Lekcja: W03E06 JavaScript I
"script" & "/script - zawiera kod JavaScript


Atrybuty HTML

Lekcja: W03E03 HTML II
-> Atrybuty są "niewidzialne" - chowamy je w znaczniku/tagu/elemencie HTML
-> Nadają elementom dodatkowe właściwości (np. "href", "src")
-> Pozwalają na grupowanie elementów (np. "class")
-> Pozwalają na identyﬁkowanie elementów (np. "id")
-> Można je łączyć (jeden znacznik może mieć wiele atrybutów)
href = "" - atrybut znacznika "a" lub "link", definiuje plik/adres do którego łączę ma prowadzić; jeśli łączę prowadzi do elementu, który znajduje się w obrębie tego samego dokumentu to nazwę elementu należy poprzedzić znakiem "#" (kotwicą)
target = "" - atrybut znacznika "a"
id = "unikalna nazwa" - identyfikuje tag/znacznik/element HTML, przydaje się do nawigacji w obrębie dokumentu HTML; może wystąpić tylko raz na stronie (każde id określa konkretny element)
src = "scieżka do pliku" - atrybut znacznika "img", definiuje obrazek (plik graficzny), który ma się pojawić na stronie (podaje ścieżkę do konkretnego pliku)
alt = "opis tekstowy obrazka" - atrybut znacznika "img", opisuje obrazek, określa jego treść

Lekcja: W0304 CSS II
style = "item: wartość; item:wartość; item:wartość; ..." - styluje konkretny tag/znacznik/element bezpośrednio w dokumencie HTML, nie poleca się stosowania tego atrybutu (ponieważ jego edycja jest bardzo kłopotliwa i czasochłonna); np. style="color: yellow; background: blue;"
class = "unikalna nazwa" - jest to prawidłowy i polecany sposób stylowania dokumentów HTML, klasy nazywamy wyłącznie po angielsku (bardzo ważne); w nazwie klasy nie może być spacji (używamy CamelCase); jedna klasa może być używana do stylowania wielu elementów; dobrą praktyką jest stylowanie absolutnie wszystkiego za pomocą klas (nawet jeśli do danej klasy jest przypisany tylko jeden element)
