! + TAB

(lekcja: W02E03 HTML I)


ALT + Z (View -> Toggle Word Wrap)

(lekcja: W02E03 HTML I)


CTRL + SHIFT + P

(lekcja: W02E03 HTML I)


SHIFT + ALT + F (left click -> Format Document)

(lekcja: W02E03 HTML I)
