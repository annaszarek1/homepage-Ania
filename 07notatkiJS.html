ES6 = ECMAScript 2015 (v6)

Lekcja: W03E06 JavaScript I
-> ECMAScript jest to ustandardyzowany przez organizację ECMA skryptowy język programowania
-> JavaScript implementuje rzeczy ze specyfikacji ECMAScript
-> Rozwój ECMAScript mocno przyspieszył dzięki szerokiemu wykorzystaniu JavaScript
-> ES6 (i nowsze wersje) wprowadzają wiele udogodnień dla deweloperów
-> Nowe funkcje mogą nie być wspierane przez wszystkie przeglądarki

Zmienne w JavaScript

Lekcja: W03E06 JavaScript I
-> Zmienna to "pudełko" na różne dane
-> Zmienne w JS są jak młody pelikan - "łykną wszystko"
-> Zmienne w JavaScript definiujemy na dwa sposoby:
1. LET - można je nadpisywać wielokrotnie w jednym dokumencie (mogą być zdefiniowane tylkok raz, ale ich wartość można zmienić);
2. CONST - można do nich przypisać wartość tylko raz w danym dokumencie (nie można ich nadpisać); opisują elementy, które mają stałą strukturę
-> W wersjach ECMAScript starszych niż 6 w użytku była zmienna "VAR"
-> Zmienne możemy definiować w dowolnym miejscu w pliku .js, jednak dobrą praktyką jest aby robić to na początku (u góry) dokumentu
